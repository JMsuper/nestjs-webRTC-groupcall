- 주제 : 실시간 위치공유 다자간 음성통화 앱(개인 프로젝트)
- 사용기술 : TypeScript, NestJS, TypeORM 0.3, Jest, WebRPC
- 완료 사항
    - ‘JWT인증’, ‘사용자’, ‘그룹참여’ 기능을 포함한 API 구현완료
        
        (로컬 환경에 한함)
        
- 진행 중인 사항
    - WebRPC 프로토콜 연구
    - 서버 구축을 위한 Janus 라이브러리 사용법 조사
- 예정 사항
    - WebRPC 테스트 방법 조사 및 수행
    - 위치공유 SaaS 조사
